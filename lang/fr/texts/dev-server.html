L'intelligence derrière les applications Open Food Facts et l'API

Open Food Facts collecte des données via une API WRITE, les traite et les rend disponible via une API READ et une version web. Computing the Nutri-Score, NOVA groups, extracting additives, regardless of their language, synonyms or potential spelling mistakes. This is just a tiny fraction of the topics covered by the server team, to ensure food transparency at planetary scale.

Contribuez au Core avec Perl

Vous voulez apprendre un nouveau langage ? Le serveur est principalement codé en Perl. Que vous soyez un vétéran accro au Perl ou que vous ayez simplement taquiné Perl pour résoudre des tâches sys-admin, vous trouverez un bug à votre taille, et des mentors pour vous guider. Et il y a plus d'une façon d'y arriver ;-)

Créez des outils utiles dans le langage de votre choix

HungerGames est un exemple d'un petit jeu codé en React qui a un impact positif sur Open Food Facts. LanguageSwitcher, un autre outil, est codé en Python. Votre outil, votre langage (mais évitez l'Assembleur ;-) )
